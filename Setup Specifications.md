### Varcel:
Vercel install: npm i -g vercel
Check if install properly: vercel --version

### Supabase: 
Run the CLI by prefixing each command with npx or bunx:

npx supabase --help
You can also install the CLI as dev dependency via npm:

npm install supabase --save-dev

### GitHub
Basic Setup & Configuration:
git init: Initializes a new Git repository in the current directory.
git config user.name "Your Name": Sets your Git username. 
git config user.email "your.email@example.com": Sets your Git email.

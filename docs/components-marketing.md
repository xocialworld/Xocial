# Marketing Components Usage

## MarketingHeader
- Renders top navigation with product and solutions links.
- Use in marketing layout; no props.

## MarketingFooter
- Renders footer links and legal.
- Use in marketing layout; no props.

## MarketingLayout
- Wraps pages with header/main/footer.
- Usage: `<MarketingLayout>{children}</MarketingLayout>`.

## Page Sections
- Use `Card` for feature highlights, `Accordion` for FAQs, `Tabs` for feature toggles.

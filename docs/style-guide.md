# Xocial Style Guide

- Typography: Tailwind defaults with extended secondary palette; headings use `text-secondary-900`.
- Colors: Primary/Secondary/Success/Info/Warning palettes; ensure contrast >= 4.5.
- Spacing: Use Tailwind spacing scale; page containers `max-w-7xl`.
- Components: Prefer ShadCN components (`Button`, `Card`, `Accordion`, `Tabs`).
- Accessibility: Landmarks (`header`, `main`, `footer`), ARIA labels on controls, keyboard focus visible.
